## 🚀 Deploy en Vercel (recomendado)

### Opción A — Drag & Drop (más fácil, sin GitHub)
1. Abrí https://vercel.com y entrá con tu cuenta
2. En el dashboard, buscá el botón **"Add New → Project"**
3. Elegí **"Browse"** o arrastrá la carpeta `fermadi-soluciones/` completa
4. Vercel detecta automáticamente que es un sitio estático
5. Hacé clic en **"Deploy"** → ¡listo en 30 segundos!
6. Te da una URL tipo `fermadi-soluciones.vercel.app`

### Opción B — GitHub + Vercel (para actualizaciones fáciles)
1. Subí la carpeta a un repo de GitHub (público o privado)
2. En Vercel: **Add New → Project → Import Git Repository**
3. Seleccioná el repo y hacé Deploy
4. Cada vez que actualices el HTML en GitHub, Vercel redeploya automáticamente

---

## 🌐 Deploy en Netlify (alternativa)
1. Abrí https://netlify.com
2. Arrastrá la carpeta `fermadi-soluciones/` al área de **drag & drop** del dashboard
3. URL lista al instante, tipo `fermadi-soluciones.netlify.app`

---

## 🔗 Dominio personalizado (opcional)
Si querés usar `fermadi.com.py` o similar:
1. Comprá el dominio en NIC.py o cualquier registrador
2. En Vercel/Netlify, ir a **Settings → Domains → Add Domain**
3. Apuntar los DNS del dominio a Vercel/Netlify (te dan las instrucciones)
